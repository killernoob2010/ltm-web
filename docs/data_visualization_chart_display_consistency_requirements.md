# 数据展示筛选与图表一致性需求

## 目标

统一数据展示页的筛选职责和图表渲染规则。筛选项变化后，图表只应改变数据范围，不应改变图表模式、图例维度或基础交互方式。

## 问题范围

- `品种池` 与高级筛选里的 `主流/非主流` 存在重复筛选。
- `品种对比图` 在筛选后只剩一个品种时，会从“品种 + 年份”图例退化成只有年份标签。
- `全品种图谱` 在筛选后只剩一个品种时，会退回普通单图画法。
- 对比图颜色按当前线条顺序分配，筛选项变化后可能导致同一条线颜色漂移。

## 筛选规则

基础筛选负责确定主要数据范围：

- `主流矿`：只显示主流矿品种。
- `非主流矿`：只显示非主流矿品种。
- `整体对比`：只显示系统聚合序列，例如 `主流矿合计`、`非主流矿合计`。
- `自定义`：允许跨主流/非主流组合。

高级筛选里的 `主流/非主流` 只在 `自定义` 品种池下生效。其他品种池下，该筛选应禁用且不参与接口请求，避免与基础筛选冲突。

## 图表规则

### 全品种图谱

- 只要用户选择 `全品种图谱`，就始终使用图谱渲染。
- 即使筛选后只剩一个品种，也保留图谱面板、年份图例和年份高亮逻辑。

### 品种对比图

- 只要用户选择 `品种对比图`，就始终使用对比图渲染。
- 图例维度固定为 `品种 + 年份`。
- 即使筛选后只剩一个品种，也保留右侧图例，不退化为只有年份标签。
- 线条颜色必须按稳定键分配，不能因为取消勾选其他品种而让剩余线条变色。

## 验收标准

- `整体对比` 取消勾选 `非主流矿合计` 后，图表只剩 `主流矿合计` 数据，但图例仍按 `主流矿合计 + 年份` 展示。
- `全品种图谱` 选剩一个品种后，仍显示单个图谱面板，而不是普通单图。
- 非 `自定义` 品种池下，高级筛选里的 `主流/非主流` 不会影响接口请求。
- 相关前端静态测试和后端筛选测试通过。

# 铁矿石基差数据与期现模块设计

## 1. 目标与交付顺序

本设计分为两个阶段：

1. 先生成一份 2024 年 1 月 1 日至今的线下基差试算 Excel，供业务核对数据、参数和计算结果；
2. Excel 口径确认后，再在数据可视化管理中增加现货/期现三级菜单、期现数据管理和期现数据展示。

第一阶段不直接接入 Web API、不修改现有数据库、不部署 Staging 或 Production。第二阶段必须复用现有数据管理宽表、筛选控件和数据展示 Canvas 折线图的布局与交互，不另做独立视觉稿。

## 2. 数据范围

- 山东：青岛港、日照港、岚山港；
- 南方：连云港（EBC 原名“连云港港”）、江阴港、太仓港；
- 唐山：京唐港、曹妃甸港。

### 2.2 品种

以 I2312 品牌规则中卡拉拉精粉及其后品种为基础，并结合现有 Web 品种口径：

- 卡拉拉精粉；
- 卡拉加斯粉；
- 乌克兰精粉；
- 昆巴粉；
- BRBF；
- 纽曼粉；
- PB粉；
- IOC6；
- 麦克粉；
- 罗伊山粉；
- 金布巴粉；
- SP10粉；
- FMG混合粉；
- 杨迪粉；
- 超特粉。

昆巴粉作为系统新增独立品种，不替换现有南非粉。

### 2.3 时间与粒度

- 起始日期：2024-01-01；
- 原始现货频率：日度；
- 基差频率：日度，仅生成现货价格和期货主力连续收盘价同时存在的交易日；
- 周末、节假日不匹配前一交易日，不生成基差；
- 港口或品种缺少现货价格时不补零、不前向填充、不使用其他港口价格替代；
- 每条日度记录同时生成业务年份、业务周次和周次标签，字段命名与现有数据管理保持一致；周次仅作为筛选和展示元数据，不做周平均。

## 3. 数据来源与职责

EBC 只提供计算所需的现货事实：指标编码、原始指标名称、港口、日期、频率、价格口径和湿吨价格。EBC 指标名称中的 Fe 不参与质量升贴水计算，只用于识别报价规格并在同品种多规格时选择高品位价格。

### 3.2 年度矿山质量参数

质量参数来源文件：

`/Users/wangjingze/建龙/期货组/本地数据库/副本进口铁矿石主要矿山货物指标-2026.xlsx`

计算日期属于哪一年，就使用工作簿相应年度页中的品种参数。质量升贴水和干湿吨换算使用年度参数中的：

- Fe；
- SiO2；
- Al2O3；
- P；
- S；
- H2O。

EBC 名称中的 Fe 不覆盖年度参数 Fe。

### 3.3 大商所规则

- 合约口径：I2312；
- 交割质量标准：F/DCE I004-2021；
- 质量升贴水：按该规则中 Fe、SiO2、Al2O3、P、S 的分段公式计算；
- 品牌升贴水：按 I2312 起实施的品牌表计算；
- 计算结果保留规则版本，避免与 I2704 及以后规则混用。

### 3.4 期货价格

- 品种：铁矿石；
- 序列：铁矿石主力连续合约；
- 价格字段：每日收盘价；
- 主力连续序列直接作为唯一盘面基准，不在本模块自行判断主力合约，不单独处理换月日；
- 只有主力连续序列存在当日收盘价时才计算当日基差；
- 明细保留期货日期、序列名称或代码、收盘价和数据来源。

## 4. 品种与参数特殊规则

### 4.1 同品种多规格价格

当 EBC 在同一港口、同一天提供同品种多个 Fe 规格时，只取高品位规格的湿吨价格：

- BRBF：优先 63%Fe；
- 金布巴粉：优先 60.3%Fe；
- 卡拉拉精粉：优先较高 Fe 规格；
- 其他品种按相同原则选择。

质量计算仍全部使用年度 Excel 参数，不使用所选 EBC 指标名称中的 Fe。

### 4.2 乌克兰精粉固定典型值

乌克兰精粉未使用年度工作表中的同名行，统一采用业务确认的典型值：

| 参数 | 百分比 | 计算存储值 |
|---|---:|---:|
| Fe | 65.54% | 0.6554 |
| SiO2 | 7.50% | 0.0750 |
| Al2O3 | 0.11% | 0.0011 |
| P | 0.07% | 0.0007 |
| S | 0.03% | 0.0003 |
| H2O | 9.00% | 0.0900 |

该组参数从 2024-01-01 起用于本次历史试算，参数来源类型记录为“业务确认典型值”。

### 4.3 Excel 缺失和格式修正规则

- 品种 S 为空时，计算值默认为 0，并记录 `s_defaulted=true`；
- PB粉 P 原值 `0.11` 按 `0.11%` 换算，计算存储值为 `0.0011`；
- 卡拉拉精粉使用年度表中的保证值，参数来源类型记录为“保证值”；
- 其他品种使用相应年度表中的典型值。

### 4.4 昆巴粉映射

昆巴粉按以下组合生成：

- 系统展示品种：昆巴粉；
- 湿吨价格：EBC“南非粉：63%Fe”港口品牌价格；
- 质量参数：年度 Excel“昆巴粉（南非）”典型值；
- 品牌规则：大商所 KUMBA 品牌升贴水；
- 元数据保留 `price_proxy_indicator=南非粉63%Fe` 和参数品种“昆巴粉（南非）”；
- EBC 没有南非粉价格的港口不计算昆巴粉。

昆巴粉与系统原有南非粉是两个独立展示品种，不合并、不互相覆盖。

### 4.5 杨迪粉映射

- 湿吨价格：EBC 杨迪粉 56.8%Fe 指标；
- 质量参数：年度 Excel 杨迪粉 58.3%典型值；
- Fe 升贴水按年度 Excel 58.3%参数计算；
- 元数据明确记录价格规格与参数规格不同。

## 5. 计算口径

### 5.1 单位统一

EBC 港口品牌价为元/湿吨，盘面价格为元/吨。现货先按年度标准水分换算为干吨：

```text
干吨现货价 = 湿吨现货价 / (1 - 年度标准水分率)
```

### 5.2 升贴水与基差

```text
质量升贴水 = Fe升贴水 + SiO2升贴水 + Al2O3升贴水 + P升贴水 + S升贴水

标准化现货价 = 干吨现货价 - 质量升贴水 - 品牌升贴水

基差 = 标准化现货价 - 铁矿石主力连续收盘价
```

正基差表示标准化现货价高于盘面收盘价，负基差表示标准化现货价低于盘面收盘价。所有价格、升贴水和基差单位统一为元/吨。

## 6. 第一阶段线下 Excel

工作簿包含四个页签。

### 6.1 期现数据

面向业务快速核对，字段保持精简并与未来数据管理一致：

- 日期；
- 周次；
- 年份；
- 港口；
- 品种；
- 湿吨现货价；
- 质量升贴水；
- 品牌升贴水；
- 主力连续收盘价；
- 基差；
- 数据状态。

### 6.2 计算明细

保存后台可查询的完整长表，包括：

- EBC 指标编码、原始名称、原始港口名、价格规格；
- 日期、业务年份、业务周次、周次标签；
- 标准品种、标准港口、湿吨价格；
- 参数年份、参数类型、Fe、SiO2、Al2O3、P、S、H2O；
- S 是否默认、价格代理指标、价格规格与参数规格是否不同；
- 干吨现货价；
- Fe、SiO2、Al2O3、P、S 分项升贴水；
- 质量升贴水合计、品牌升贴水、标准化现货价；
- 期货日期、主力连续序列、收盘价；
- 基差、规则版本、计算状态和备注。

### 6.3 参数表

汇总年度品种参数、乌克兰精粉固定典型值、S 缺失默认规则、PB粉 P 换算规则、卡拉拉保证值、昆巴粉和杨迪粉特殊映射、大商所品牌升贴水与规则版本。

### 6.4 数据质量说明

按品种、港口和时间记录：

- 源数据不存在的港口-品种组合；
- 指标停更；
- 低流动性或连续性维护说明；
- 年度参数缺失或格式修正；
- S 默认 0；
- 代理价格或价格规格与参数规格不一致。

该页只用于审计说明，第一版 Web 不增加异常统计卡片。

## 7. 后续 Web 信息架构

三级导航统一使用“期现”，不使用“期限”：

```text
数据管理
├── 现货数据管理
└── 期现数据管理

数据展示
├── 现货数据展示
└── 期现数据展示
```

- 现货数据管理和现货数据展示承接当前页面内容与行为；
- 期现数据管理展示基差日度精简字段，布局参考现有数据管理宽表；
- 期现数据展示第一版展示基差，后续可在同一模块扩展其他期现指标；
- 第一版不提供 Excel 导出、独立完整计算明细查询、异常统计卡片或额外趋势分析功能；
- 完整计算明细保存在后台长表，支持直接数据库查询。

## 8. 期现数据管理与展示

### 8.1 筛选项

期现数据管理和期现数据展示只保留：

- 品种；
- 年份；
- 港口。

筛选选项从实际计算结果动态生成，不展示无数据组合。

### 8.2 图表

- 按品种分类；
- 每个品种分类下展示各港口基差；
- 支持现有图表的多年份区分和筛选逻辑；
- 每个实际交易日绘制一个日度数据点；
- 横轴显示月份，底层仍保留完整日期；
- 悬浮提示显示具体日期、年份、港口和基差；
- 缺失日期不补点、不补零、不生成虚假连续值；
- 布局、颜色、图例、高亮、响应式方式与当前数据展示图表保持一致。

## 9. 数据标识与唯一性

### 9.1 原始现货事实

业务唯一键：

```text
EBC指标编码 + 数据日期
```

### 9.2 基差结果

业务唯一键：

```text
数据日期 + 标准港口 + 标准品种 + 规则版本 + 参数版本
```

后续数据库使用独立技术主键 `id`。EBC 指标编码、原始指标名、参数版本和规则版本必须保留用于审计，不把铁含量写入品种名称或业务唯一键。

## 10. 验收标准

### 10.1 线下 Excel

- 覆盖 2024-01-01 至提取日；
- 覆盖 8 个目标港口及所有有源数据的目标品种；
- 同品种多规格只采用高品位 EBC 价格；
- 年度参数、特殊映射和默认值按本设计执行；
- 只在主力连续收盘价存在的日期计算基差；
- 随机抽查 BRBF、PB粉、杨迪粉、昆巴粉、乌克兰精粉和卡拉拉精粉，公式及来源均可追溯；
- 期现数据与计算明细的记录数、基差结果一致；
- 无公式错误、日期错配、百分比单位错误、零填充或前向填充；
- 每个页签完成可读性和视觉检查。

### 10.2 后续 Staging Web

- 原有现货数据管理和展示功能不回归；
- 现货/期现三级菜单名称和跳转正确；
- 期现数据管理筛选、表格和数据量与已确认 Excel 一致；
- 期现数据展示按品种、港口、年份绘制日度基差，横轴显示月份；
- 缺失组合和非交易日不产生数据点；
- 页面复用现有视觉与交互，没有裁切、重叠、空白图表或控制台错误；
- Production 不在第一阶段和未单独确认的第二阶段授权范围内。

## 11. 非目标

- 不计算港口库存实际化验质量基差；
- 不导入或估算缺失港口价格；
- 不生成周平均、月平均或周度基差；
- 不自行拼接期货主力合约或处理换月日；
- 不在第一版 Web 提供导出、人工改值、异常统计卡片或完整计算明细页面；
- 不采用 I2704 及以后规则。

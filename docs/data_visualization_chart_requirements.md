# 数据可视化管理 - 数据展示模块需求调整

## 目标

数据展示模块从“多筛选项堆叠的万能图表”调整为“按数据种类组织的品种图谱系统”。核心目标是减少默认筛选噪音，让用户围绕一个数据种类快速比较多个品种在不同年份的走势。

## 数据类型

整合结果中的 `数据类型` 需要拆分为四类：

```text
发运
到港
库存
表需
```

四类含义：

- `发运`：全球或产地口径的发运数据，不等同于中国港口可用量。
- `到港`：中国港口或中国锚地口径的到港数据；也包含由发运按比例、滞后等规则折算出的到港估算。
- `库存`：中国港口库存。
- `表需`：通过中国口径 `到港 + 上期库存 - 本期库存` 计算出的需求值。

表需计算只使用 `到港` 与 `库存`，不得再直接使用 `发运` 字段。

## 整合明细建议字段

现有字段继续保留：

```text
统计周一
统计周日
业务年份
业务周次
周次标签
展示日期
数据类型
来源/国家
品种
种类
主流/非主流
数值
单位
来源文件
来源Sheet
来源区域
是否参与表需
校验状态
备注
```

后续建议新增审计字段：

```text
数据口径
生成方式
折算说明
对比分组
```

字段含义：

- `数据口径`：全球发运、中国到港、中国港口库存、中国表需等。
- `生成方式`：原始数据、发运折算到港、系统计算。
- `折算说明`：例如“巴西中国大陆发运量 * 75%，滞后 6 周”。
- `对比分组`：主流矿、非主流矿、主流矿合计、非主流矿合计等。

第一阶段可以先把这些信息写入 `备注`，但数据库和 Excel 字段设计应预留向独立字段迁移的空间。

## 页面结构

数据展示模块保留四个数据种类页签：

```text
发运图表
到港图表
库存图表
表需图表
```

用户通常不会把发运、到港、库存、表需四类放在同一张图里比较，因此默认每次只展示一个数据种类。

每个数据种类下提供两个视图模式：

```text
全品种图谱
品种对比图
```

### 全品种图谱

全品种图谱用于快速浏览一个数据种类下多个品种的走势。

展示规则：

- 每个品种一张小图。
- 每张小图里按年份画多条线。
- 所有小图共享年份颜色。
- 点击任意小图中的年份后，所有小图同步高亮该年份。
- 默认展示当前年份和近几年历史年份。
- 鼠标悬停在高亮年份节点上显示日期、周次、年份、品种、数值。

当选择品种数量较多时，默认使用全品种图谱，避免单图线条过多。

### 品种对比图

品种对比图用于主动选择几个品种放在同一张大图里对比。

展示规则：

- 横轴为业务周。
- 纵轴为当前数据种类的数值。
- 颜色优先表示年份。
- 同一年不同品种使用线型或节点样式区分。
- 当前年份默认加粗并显示节点。
- 选中品种数量建议 2-6 个；超过 6 个时提示切换到全品种图谱更清晰。

## 品种池

品种选择不再默认平铺所有品种。先选择品种池，再显示对应品种：

```text
主流矿
非主流矿
整体对比
自定义
```

规则：

- `主流矿`：只显示主流矿品种。
- `非主流矿`：只显示非主流矿品种。
- `整体对比`：显示系统生成的整体序列，如主流矿合计、非主流矿合计。
- `自定义`：允许跨主流/非主流混选，用于临时分析。

默认不把主流矿和非主流矿混在同一个品种列表里。

## 系统计算序列

系统需要支持以下虚拟品种或聚合序列：

```text
主流矿合计
非主流矿合计
```

后续可扩展：

```text
主流粉矿合计
非主流粉矿合计
块矿合计
球团合计
精粉合计
```

这些序列用于整体对比，不要求用户手动勾选多个品种后再临时判断。

## 筛选设计

默认常驻筛选只保留：

```text
数据种类
视图模式
品种池
品种
年份
```

以下筛选放入“高级筛选”，默认折叠：

```text
来源/国家
种类
主流/非主流
数据口径
生成方式
```

高级筛选不得再用大量 checkbox 平铺，应使用可搜索多选或折叠式标签。

## 颜色与交互

年份颜色在所有图中保持一致。建议固定近几年颜色：

```text
2022 蓝
2023 橙
2024 灰
2025 黑或黄
2026 红
```

交互规则：

- 点击年份：全局高亮该年份。
- 再次点击同一年：取消高亮。
- 高亮年份：线条加粗、色彩饱和、显示节点。
- 非高亮年份：降低透明度。
- 鼠标悬停：展示日期、周次、年份、品种、数值。

## 验收标准

- 数据展示支持 `发运 / 到港 / 库存 / 表需` 四类页签。
- `发运` 与 `到港` 在数据和界面上明确拆分。
- 表需数据只由到港和库存口径产生或读取。
- 默认页面不再显示所有筛选项，常驻筛选控制在 5 个以内。
- 支持全品种图谱和品种对比图两种视图模式。
- 支持主流矿、非主流矿、整体对比、自定义四种品种池。
- 支持多品种对比，主流矿和非主流矿默认不混选。
- 年份颜色在不同小图之间保持一致，点击年份后可跨图高亮。
